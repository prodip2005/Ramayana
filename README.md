<p>
<b>Valmiki Ramayana</b> is a full-stack web application designed for spiritual seekers and scholars to explore the epic Ramayana shlokas with ease. Users can navigate through different Kandas and Sargas, read divine verses, and bookmark their favorite shlokas for future reference in their personal profile.

<h2>ğŸ› ï¸ Installation & Local Setup</h2>

<ol>
  <li>Clone the repositories:
    <pre># Clone Client
git clone https://github.com/prodip2005/Ramayana.git

# Clone Server
git clone https://github.com/prodip2005/Ramayana-Server.git</pre>
  </li>
  <li>Install dependencies:
    <pre># For Client
cd Ramayana && npm install

# For Server
cd ../Ramayana-Server && npm install</pre>
  </li>
  <li>Environment Variables:
    <ul>
      <li>In <b>Ramayana-Server</b> folder, create a <code>.env</code> file for your <code>MONGODB_URI</code>.</li>
      <li>In <b>Ramayana</b> (Client) folder, create a <code>.env.local</code> for Firebase Config keys.</li>
    </ul>
  </li>
  <li>Start Development:
    <pre># Start Backend (Inside Ramayana-Server)
npm start

# Start Frontend (Inside Ramayana)
npm run dev</pre>
  </li>
</ol>
